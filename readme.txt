== Description ==

Add Top bar on WordPress Site.


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/seed-top-bar` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

There is no settings.
